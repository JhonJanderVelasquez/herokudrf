REST API CON VIEWSET PARA ZAITO

INSTRUCCIONES PARA EL MANEJO DE LA API

Las urls disponibles para consultar en la API son:
El respositorio de esta API lo puedes encontrar aquí.
1. https://apirestzaito.herokuapp.com/api/products/
2. https://apirestzaito.herokuapp.com/api/images/

3. https://apirestzaito.herokuapp.com/api/products/{id}
4. https://apirestzaito.herokuapp.com/api/images/{id}

  • Si deseas por ejemplo consultar todos los productos, usas la 1ra url.
  • Si deseas consultar solo la información de un producto, entonces la 3ra url.
  • Si deseas realizar un POST para crear un nuevo producto, envías la petición por la 1ra url.
  • Si deseas realizar un DELETE de un producto, lo haces a través de la 3ra url.
    • Y así sucesivamente...

Si deseas consultar todos los productos de la base de datos de la API, entonces da clic sobre el carrito que está en la parte izquierda.

Si deseas consultar todos las imágenes de la base de datos de la API, entonces da clic sobre la cámara que está a la derecha.
