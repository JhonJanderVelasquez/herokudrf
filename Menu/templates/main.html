REST API CON VIEWSET PARA ZAITO

INSTRUCCIONES DE LA API:

El repositorio de esta API lo puedes encontrar aquí.

1apirestzaito.herokuapp.com/api/products/
2apirestzaito.herokuapp.com/api/images/
3apirestzaito.herokuapp.com/api/products/{id}
4apirestzaito.herokuapp.com/api/images/{id}
INSTRUCCIONES API POSTMAN PARA PRODUCTOS
  • Si deseas consultar todos los productos, usas la 1ra url. Este es un método List.
  • Para consultar solo la información de un producto, usas la 3ra url. Este es un método de tipo Retrieve.
  • Para crear un nuevo producto, envías la petición en formato JSON usando la 1ra url. Este es un método de tipo Create
  • Para eliminar un producto, lo puedes hacer a través de la 3ra url. Este es un método de tipo Destroy
  • Si deseas actualizar un producto, lo puedes hacer a través de la 3ra url. Este es un método de tipo Update
Si deseas detallar la documentación tipo Swagger para conocer todos las urls permitidas por la API, entonces da clic sobre el libro que está a la derecha.
Si deseas consultar todos los productos de la base de datos de la API, entonces da clic sobre el carrito que está en la parte izquierda.
Si deseas consultar todos las imágenes de la base de datos de la API, entonces da clic sobre la cámara que está a la derecha.
